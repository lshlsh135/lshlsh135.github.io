Claude가 Amazon Bedrock에서 정식 출시(GA)되어 이제 모든 AWS 고객이 쓸 수 있게 됐어요. Bedrock은 AWS가 운영하는 풀매니지드 서비스로, 파운데이션 모델과 생성형 AI 앱 빌딩 도구에 대한 보안 클라우드 접근을 제공해요. 즉, 별도의 인프라 구축 없이 AWS 환경 안에서 Claude를 호출해 생성형 AI 애플리케이션을 만들 수 있다는 뜻이에요. 기존에 AWS 보안·결제·IAM 체계를 그대로 쓰면서 Claude를 도입하고 싶은 팀에게 의미가 큰 발표예요.
