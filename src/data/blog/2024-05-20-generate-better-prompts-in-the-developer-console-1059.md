Anthropic Console에 프롬프트 템플릿 자동 생성 기능이 들어왔어요. Claude가 바로 쓸 수 있는 수준의 프롬프트 템플릿을 만들어 줘서, 개발자가 처음부터 프롬프트를 짜느라 시간을 쓰지 않아도 된다는 게 핵심이에요. 콘솔에서 곧장 'production-ready' 템플릿을 뽑을 수 있다고 소개되어 있고요. 프롬프트 엔지니어링에 익숙하지 않거나, 빠르게 베이스라인 프롬프트를 만들어 두고 거기서부터 튜닝해 나가고 싶은 개발자에게 유용해 보입니다. (구체적인 UI 위치나 옵션은 원문에 상세 설명이 없어, 실제 동작과 한계는 콘솔에서 직접 확인하는 게 정확해요.)
