Claude가 외부 도구와 API에 직접 연결돼 작업을 수행하고 데이터를 다루며 더 정확한 답변을 낼 수 있게 됐어요. 이번 발표로 tool use 기능이 Claude 3 모델 패밀리 전체에 걸쳐 정식 출시(GA)됐고, Anthropic Messages API와 Amazon(베드락 추정) 등 주요 채널에서 사용할 수 있습니다. 핵심은 모델이 호출할 수 있는 외부 도구를 정의해두면, Claude가 상황에 맞춰 그 도구를 호출하고 결과를 받아 응답에 반영한다는 점이에요. 즉 단순 텍스트 생성에서 벗어나 실제 시스템과 상호작용하는 에이전트형 사용이 공식적으로 가능해진 셈입니다. 구체적인 스펙·예제 코드는 본문이 비어 있어 확인되지 않으니, 적용 전 Anthropic 공식 문서에서 최신 파라미터와 지원 모델 범위를 직접 확인하세요.
