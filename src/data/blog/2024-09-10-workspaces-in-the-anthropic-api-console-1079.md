Anthropic이 API 콘솔에 워크스페이스(Workspaces) 기능을 도입했어요. 한 계정 안에서 여러 Claude 배포(프로젝트, 환경, 팀)를 따로따로 관리할 수 있게 해주는 게 핵심이에요. 본문이 제공되지 않아 세부 권한 모델이나 한도 설정 방식 같은 구체적인 운영 디테일까지는 확인되지 않지만, 발표 취지는 '여러 배포를 효율적으로 관리'하는 데 있어요. 여러 서비스/환경에서 Claude를 동시에 굴리는 개발자나 팀이라면 콘솔에서 한번 들여다볼 만한 업데이트예요.
