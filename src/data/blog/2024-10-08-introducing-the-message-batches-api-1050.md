Anthropic이 대량 쿼리를 비동기로 처리하는 Message Batches API를 공개했어요. 동기 호출 대비 더 저렴한 비용으로 한 번에 큰 분량의 요청을 묶어 보낼 수 있는 게 핵심이에요. 업데이트로는 Anthropic API에서 정식 출시(GA)되었고, Amazon Bedrock에서 Claude를 쓰는 고객은 배치 추론(batch inference) 형태로 활용할 수 있다고 해요. 즉시 응답이 필요 없는 평가, 분류, 대량 데이터 처리 같은 작업에 비용을 아끼고 싶을 때 고려해볼 만한 옵션이에요.
