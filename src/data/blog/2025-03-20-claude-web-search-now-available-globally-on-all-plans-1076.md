클로드가 인터넷을 직접 검색해서 최신 정보를 인용과 함께 답변하는 웹 검색 기능이 이제 전 세계 모든 요금제에서 쓸 수 있게 됐어요. 그동안 일부 지역·플랜에 한정됐던 기능이 글로벌 확장된 거라서, 무료 사용자도 동일한 실시간 검색 기반 답변을 받을 수 있는 게 핵심이에요. 사용 사례로는 리서치, 시장 분석처럼 최신 데이터가 중요한 작업이 명시돼 있고, 답변에는 출처 인용이 따라붙어서 사실 확인이 쉬워요. 학습 데이터 컷오프 이후의 사건이나 빠르게 바뀌는 정보를 다룰 때 더 이상 '모르겠다' 답변에 막히지 않아도 된다는 점이 실질적 변화예요.
