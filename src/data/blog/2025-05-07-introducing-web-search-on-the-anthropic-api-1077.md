Anthropic API에서 Claude가 웹을 직접 검색할 수 있게 됐습니다. 이제 API를 통해 실시간 정보를 가져올 수 있고, 응답에는 출처 인용(citations)이 함께 제공돼 최신 정보 기반 AI 앱을 만들기 좋아졌습니다. 별도의 외부 검색 도구를 붙이지 않아도 Claude 호출 안에서 웹 검색 흐름을 처리할 수 있다는 게 핵심입니다. 다만 본문이 제공되지 않아 구체적인 파라미터, 가격, 지원 모델, 도메인 필터링 옵션 등은 이 글만으로는 알 수 없으니 실제 적용 전에 Anthropic 공식 문서를 확인하는 게 안전합니다.
