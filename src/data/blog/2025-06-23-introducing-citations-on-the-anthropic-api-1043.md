Anthropic이 Claude가 답변할 때 원본 문서의 특정 구절을 인용할 수 있도록 해주는 Citations 기능을 API에 새로 출시했습니다. 모델이 응답을 소스 문서에 근거(ground)시키면서 어디서 가져온 정보인지 출처를 함께 추적해주기 때문에, 답변의 근거를 사람이 직접 검증할 수 있게 됩니다. 2025년 6월 30일 업데이트로 Amazon Bedrock에서도 이 기능을 쓸 수 있게 됐습니다. RAG·문서 Q&A·법률·리서치처럼 출처 신뢰성이 중요한 워크플로에서 환각을 줄이고 사후 검증 비용을 낮추는 용도로 활용하기 좋아 보입니다.
