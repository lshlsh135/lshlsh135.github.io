Anthropic이 Enterprise와 Team 고객을 위해 Claude Code를 프리미엄 시트(premium seats) 형태로 제공하기 시작했습니다. 함께 추가된 것은 관리자 컨트롤, 사용량 분석(usage analytics), 그리고 거버넌스를 위한 Compliance API 세 가지입니다. 즉 조직 차원에서 누가 Claude Code를 쓰는지 통제하고, 사용 현황을 모니터링하며, 컴플라이언스 데이터를 시스템적으로 받아갈 수 있는 길이 열린 셈입니다. 비즈니스 플랜에서 Claude Code 도입을 검토하던 IT·보안·관리자에게 의미 있는 업데이트로 보입니다. 다만 EXCERPT 외 본문이 제공되지 않아 구체적 시트 가격, API 엔드포인트, 분석 항목 등은 이 글만으로는 확인할 수 없습니다.
