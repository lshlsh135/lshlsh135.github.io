마이크로소프트가 Microsoft 365 Copilot에 Claude 모델을 정식으로 추가했어요. 이제 Copilot을 쓰는 수백만 명의 기업 사용자들이 자기 워크스페이스 안에서 바로 Claude를 호출할 수 있다는 뜻이에요. MS 생산성 플랫폼이라는 큰 유통 채널을 통해 Claude가 들어가는 거라, 그동안 사내 정책상 Anthropic API를 직접 못 쓰던 조직도 Copilot 라이선스만으로 Claude를 합법적으로 활용할 길이 열렸다는 점이 핵심이에요. 다만 어떤 Claude 모델이 어떤 Copilot 기능에 연결되는지, 사용자별 선택 UI가 어떻게 되는지 같은 세부 사항은 이 발표문 본문이 비어 있어 확인할 수 없으니 Microsoft / Anthropic 공식 공지를 함께 참고하세요.
