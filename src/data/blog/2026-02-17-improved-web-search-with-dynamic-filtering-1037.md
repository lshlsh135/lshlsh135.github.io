Claude의 웹 검색·웹 페치 도구가 이제 코드 실행을 활용해 검색 결과를 컨텍스트 윈도에 들어가기 전에 동적으로 필터링한다. 벤치마크 기준으로 정확도는 평균 11% 올라가고 입력 토큰은 24% 더 적게 쓴다고 한다. API에서 바로 쓸 수 있어서, 검색 도구를 붙인 에이전트를 운영 중이라면 별도 마이그레이션 없이 품질·비용 양쪽에서 이득을 볼 가능성이 높다.
