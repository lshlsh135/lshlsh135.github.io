Claude Code 데스크톱 앱이 코드 작성에서 머지까지의 흐름을 한 곳에서 처리할 수 있도록 업데이트됐다는 소식이다. 실행 중인 앱을 바로 미리보고, 변경된 diff를 자동으로 리뷰하며, PR 상태까지 모니터링하는 기능이 데스크톱에 통합됐다고 한다. 별도 도구로 창을 옮겨다닐 필요 없이 작성·검토·머지를 한 자리에서 끝낼 수 있다는 점이 핵심이다. 본문이 제공되지 않아 구체적인 UI 위치나 명령은 확인되지 않으며, 자세한 사용법은 원문/공식 안내를 참고하는 편이 정확하다.
