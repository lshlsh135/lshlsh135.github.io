Anthropic이 skill-creator 도구를 업데이트해서 Agent Skills 작성자가 자기 스킬이 실제로 잘 작동하는지 검증하고, 회귀(regression)를 잡아내고, 스킬 설명문을 개선할 수 있도록 했다. 핵심은 '코딩 없이도' 이 모든 게 가능하다는 점이라, 비개발자 스킬 저자도 품질 관리 루프를 돌릴 수 있게 됐다. 스킬을 만들고 끝이 아니라 '테스트하고 측정하고 다듬는' 사이클이 정식으로 지원된다는 신호로 읽으면 된다. 본문이 제공되지 않아 구체적인 명령어나 측정 지표는 확인할 수 없으니, 실제 사용법은 원문에서 확인하는 게 안전하다.
